Online Bedava Browser Oyunları

Yüklemeden direkt internet tarayıcınızdan oynayabileceğiniz oyun linkleri.

  • VimGolf

    Vim'de yetenek geliştirmek için çeşitli bulmacalar

    https://www.vimgolf.com/
  • Vim Adventures

    Vim öğrernmek için geliştirilmiş oyun

    https://vim-adventures.com/
  • Shapez.io

    Javascript ile yazılmış Factorio benzeri oyun

    https://shapez.io/
  • Design Patterns Game

    Javascript ile yazılmış tasarım desenlerini tanıma üzerine bir oyun

    https://designpatternsgame.com/
  • Untrusted

    Oyunun kodları üzerinde değişiklik yaparak seviyeleri geçmeniz gereken roguelike oyun

    http://alexnisnevich.github.io/untrusted/
  • A Dark Room

    Çok basit başlayan ancak sonrasında inanılmaz derinliğe ulaşan metin macera oyunu

    http://adarkroom.doublespeakgames.com/?lang=tr
  • Hextris

    Tetris benzeri bulmaca-aksiyon oyunu

    https://hextris.io/
  • Quick Draw

    Yapay zeka ile karşılıklı oynanan bir oyun. Size çıkan kelimeyi ekrana çiziyorsunuz eğer yapay zeka çiziminizden kelimeyi bilebilirse puan kazanıyorsunuz. Tam oyun da denemez deney gibi bir şey.

    https://quickdraw.withgoogle.com/
  • Fluke Out

    CSS seçicilerin öğrenmenizi sağlayan bir oyun. Ön yüz geliştiricileri için şiddetle tavsiye ediyorum.

    https://flukeout.github.io/
  • Elevator Saga

    Javascript ile bir asansörün davranışlarını kodlama oyunu. Her aşamada asansörün davranışlarını değiştirerek hedef sayıdaki kişiyi istediği kata götürmeye çalışıyorsunuz.

    http://play.elevatorsaga.com/
  • FlexBox Defense

    CSS'deki FlexBox'u öğretmek yazılmış bir kule defans oyunu. FlexBox özelliklerini kullanarak kulelerinizin yerini ve yönünü belirliyorsunuz.

    http://www.flexboxdefense.com/
